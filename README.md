# Logistic Regression Classification

This project demonstrates a complete Machine Learning classification pipeline using Logistic Regression with Scikit-learn.

## Features
- Synthetic dataset generation
- Data preprocessing and normalization
- Logistic Regression model training
- Performance evaluation using:
  - Accuracy
  - Precision
  - Recall
  - F1 Score
  - ROC-AUC
- Confusion Matrix visualization
- ROC Curve plotting

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow
1. Generate dataset
2. Split training/testing data
3. Standardize features
4. Train Logistic Regression model
5. Evaluate model performance
6. Visualize metrics and ROC curve
